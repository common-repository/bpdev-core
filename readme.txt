Contains core functions of BPDEV Plugins ( a set of BuddyPress plugins ).

== Description ==

Not shown in the admin area, this is key to make everything working. It contains functions, filters and actions that are used to make all the BPDEV functional, that's way it's called Core.

BPDEV Core contains:

* BPDEV Framework ( a framework to write BuddyPress Plugins )
* BPDEV Admin ( to manage BPDEV Plugins )
* BPDEV SimplePie ( that loads SimplePie library in MU )

== Installation ==

Install BPDEV Core is really simple

1. Upload the `bpdev-core` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress as Site Wide
3. Read the license.txt
4. Start playing with the 'BPDEV Admin' menu in the WordPress backend
